# 黄金预测功能说明

## 概述

新增了黄金价格预测功能，作为对股票和比特币预测的补充。现在支持三种数据源：

1. **股票** (akshare) - A股、港股、美股等
2. **比特币** (yfinance) - BTC/USDT 加密货币
3. **黄金** (yfinance) - GC=F 黄金期货

## 使用方法

运行预测脚本时，选择数据源选项：

```
请选择数据源（1=股票[akshare], 2=比特币[yfinance], 3=黄金[yfinance]，默认1）:
```

输入 `3` 选择黄金预测。

## 黄金数据说明

- **数据源**: Yahoo Finance (yfinance)
- **交易代码**: GC=F (COMEX黄金期货)
- **支持周期**: 日线、分钟线
- **数据字段**: 开盘价、最高价、最低价、收盘价、成交量、成交额、时间戳

## 测试功能

可以使用以下命令测试黄金数据获取功能：

```bash
python prediction_example.py --test-gold
```

## 注意事项

- 黄金数据通过Yahoo Finance获取，可能受到网络和数据可用性影响
- 建议使用日线数据获得更稳定的预测结果
- 黄金期货交易时间通常为周一至周五，需要注意节假日数据缺失

## 技术实现

- 新增 `get_gold_data()` 函数，类似比特币数据获取逻辑
- 使用 yfinance 库获取 GC=F 黄金期货数据
- 支持重试机制和数据验证
- 与现有股票和比特币预测流程完全兼容
